Why Windfall?

Free & Open Source

MIT licensed. Full source code available on GitHub. No hidden costs, no paywalls.

33 Checks

Comprehensive detection covering movement, combat, inventory, and packet exploits.

Low Impact

Async processing and optimized packet listeners keep your TPS high and latency low.

Modern MC Versions

Updated for Mojang's year-based versioning. Supports 26.x through 26.2 out of the box.

Simple Setup

Drop the JAR in, add PacketEvents, and you're running. No database required.

Paper & Spigot

Works on both Paper and Spigot servers with no extra configuration needed.

Quick Start

  1. Download the latest release
  2. Install PacketEvents 2.13.0 as a separate plugin
  3. Drop both JARs into your plugins/ folder
  4. Restart your server
/windfall reload
